Slack 기본 기능 가이드

메시지 보내기

#
채널 (Channel)

주제별 대화방. 팀 전체가 볼 수 있어요.

  • #general — 전체 공지
  • #random — 자유로운 잡담
  • 채널명은 소문자, 하이픈(-) 사용 권장
  • 공개(Public) / 비공개(Private) 설정 가능
✉
다이렉트 메시지 (DM)

1:1 또는 소그룹 비공개 대화

  • 왼쪽 사이드바 → "다이렉트 메시지" 클릭
  • 최대 9명까지 그룹 DM 가능
  • 채널과 달리 이름이 없고 멤버 목록으로 표시
↩
스레드 (Thread)핵심

메시지에 답글을 묶어 대화를 정리해요.

  • 메시지에 마우스 올리기 → "답글 달기" 클릭
  • 채널이 지저분해지지 않게 해주는 핵심 기능
  • 스레드 답글은 채널에 별도 알림 없이 진행

알림 & 멘션

@
멘션 (@mention)

특정인에게 알림을 보낼 때 사용해요.

  • @이름 — 특정 사람 호출
  • @channel — 채널 전체 멤버에게 알림
  • @here — 현재 접속 중인 멤버에게만 알림
  • 꼭 필요할 때만 사용하는 게 에티켓
🔔
알림 설정

받고 싶은 알림만 골라서 설정할 수 있어요.

  • 프로필 사진 클릭 → "환경설정" → "알림"
  • 채널별로도 개별 알림 설정 가능
  • 방해 금지 시간(Do Not Disturb)도 설정 가능
😊
상태 설정

자리비움, 회의 중 등 현재 상태를 표시해요.

  • 프로필 사진 클릭 → "상태 업데이트"
  • 이모지 + 텍스트로 자유롭게 설정
  • 자동 만료 시간 설정 가능 (예: 1시간 후)

유용한 기능

👍
이모지 반응 (Reaction)자주 씀

메시지에 이모지로 가볍게 반응할 수 있어요.

  • 메시지에 마우스 올리기 → 이모지 아이콘 클릭
  • "확인했어요" 를 말 대신 👍로 전달 가능
  • 채널 알림 없이 조용히 반응할 수 있어 유용
🔍
검색

지난 메시지와 파일을 빠르게 찾아요.

  • 상단 검색창 또는 Ctrl+K (Mac: ⌘+K)
  • 특정 채널 내 검색: in:#채널명 검색어
  • 특정 사람 메시지: from:@이름 검색어
🔖
저장 (나중에 보기)

중요한 메시지를 저장해 두고 나중에 확인해요.

  • 메시지에 마우스 올리기 → 북마크 아이콘 클릭
  • 왼쪽 사이드바 → "나중에 보기" 에서 모아볼 수 있어요

각 카드를 클릭하면 상세 내용을 볼 수 있어요

Slack 알림 규칙 및 최신 콘텐츠 확인 방법

항상 알림 옴

알림
나를 @멘션한 메시지
누군가 @내이름 으로 나를 직접 호출할 때
알림
@channel / @here 메시지
채널 전체나 현재 접속자에게 보내는 단체 알림
알림
내가 보낸 스레드에 답글
내가 직접 작성한 메시지의 스레드에 누가 답글을 달 때
알림
내가 참여한 스레드에 답글
내가 한 번이라도 답글을 달았던 스레드에 새 답글이 올 때
단, 스레드 알림을 직접 끄면 더 이상 오지 않아요
알림
DM (다이렉트 메시지)
나에게 직접 보내는 1:1 또는 그룹 DM

조건부 / 설정에 따라 다름

조건부
내가 태그된 글에 달린 댓글 — 핵심 질문!
누가 내 메시지를 멘션하고 다른 사람이 그 스레드에 답글을 달면,
내가 그 스레드에 참여한 것이 되어 알림이 와요.
단, 내가 스레드 알림을 수동으로 끈 경우는 알림 안 옴
조건부
채널 일반 메시지
채널 알림을 "모든 새 메시지"로 설정한 경우에만 알림 옴
기본값은 멘션만 알림

알림 안 옴

알림 없음
내가 참여하지 않은 스레드의 답글
채널에 있는 다른 사람들 스레드 대화는 알림 없음
알림 없음
이모지 반응 (reaction)
내 메시지에 누가 👍 이모지를 달아도 기본적으로 알림 없음

모바일 "따라잡기" 외에도, 데스크탑과 모바일 모두에서 최신 내용을 확인하는 방법이 여러 가지 있어요.

🏠
홈 탭 (Activity) 데스크탑 + 모바일
왼쪽 사이드바 맨 위 집 모양 아이콘 → 나와 관련된 최신 활동을 모아서 보여줘요. 멘션, 답글, 반응이 한 곳에 모여요.
경로: 사이드바 → 홈 아이콘 (Home)
@
멘션 & 반응 데스크탑 + 모바일
나를 태그한 메시지, 내 메시지에 달린 이모지 반응만 골라서 볼 수 있어요. 놓친 멘션을 빠르게 확인할 때 유용해요.
경로: 사이드바 → @ 아이콘 (멘션 & 반응)
↩
스레드 모아보기 데스크탑 + 모바일
내가 참여하거나 팔로우하는 모든 스레드의 새 답글을 한 번에 확인할 수 있어요.
경로: 사이드바 → 스레드 아이콘 (Threads)
★
읽지 않은 메시지만 보기 데스크탑
채널 목록에서 굵게 표시된 채널들이 읽지 않은 메시지가 있는 곳이에요. 사이드바 상단 "읽지 않은 메시지" 섹션에서 한 번에 볼 수도 있어요.
경로: 사이드바 상단 → 읽지 않은 메시지 (Unreads)
